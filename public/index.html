How It Works

Add the Script

Drop a single <script> tag anywhere on your page. No npm, no build step, no configuration.

→

→

Drag, Drop, Type

Everything You Need

Professional visual editing, right in the browser

Inline Text Editing

Click any text element to edit in place. Bold, italic, links, colors, heading levels — all via a floating toolbar.

Drag & Drop Layout

Grab the ⠿ handle to reorder any block element. A glowing blue line shows exactly where it'll land.

Style Panel

Adjust colors, padding, margin, border radius, opacity, CSS classes, and raw CSS — all from a sleek side panel.

Image Controls

Replace images via URL, edit alt text, align with one click, and drag corner handles to resize visually.

Undo / Redo

Full history stack. Ctrl+Z / Cmd+Z to undo, Ctrl+Shift+Z to redo. Every change is tracked.

Export Options

Save clean HTML, copy to clipboard, or get a JSON diff of exactly what changed — selector, before, and after.

Non-Destructive

Zero impact when not editing. No external CSS. All editor UI injected via JS and namespaced with n3-.

Mobile Ready

Touch-friendly drag and drop. Works on iOS and Android. Editor UI adapts to any screen size.

Zero Dependencies

Pure vanilla JavaScript. Works on WordPress, static HTML, React apps — any website. No npm, no bundler.

Clean, modular internals

10-Class Modular Architecture

A single file, internally organized into clean, decoupled classes. Event-driven communication. Dependency injection. JSDoc throughout.

Core
N3Editor— main orchestrator, public API surface
N3Events— pub/sub bus: on() emit() off() clear()
N3History— undo/redo stack with canUndo/canRedo
N3UI— CSS injection, toast, confirm dialog
N3Export— cleanHTML(), diff(), download, copy
Editing
N3TextEditor— contentEditable + formatting toolbar
N3DragManager— drag start/move/end, drop indicator
N3ElementControls— hover overlays, dup, delete
N3StylePanel— right sidebar: colors, spacing, classes
N3Toolbar— top bar: undo/redo, save, exit
Advanced API
n3ware.on('editor:modeChange', cb) // event bus
n3ware.on('drag:drop', cb) // drag events
n3ware._modules.history.canUndo() // direct access
n3ware._modules.exporter.diff(a, b) // diff API
