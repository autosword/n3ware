Everything you need to edit

A complete visual editing suite packed into a single file with zero dependencies.

✏️

Inline Text Editing

Click any heading, paragraph, link, or list item to edit directly on the page. Floating formatting toolbar with bold, italic, underline, alignment, and font size controls.

⠿

Drag & Drop Layout

Grab the drag handle on any block element to reorder it. A crisp blue insertion line shows exactly where your element will land before you drop it.

🎨

Live Style Panel

Select any element to open the right-side style panel. Adjust background color, text color, padding, margin, border radius, opacity, and CSS classes — all in real time.

📋

Element Controls

Hover any block to reveal type label, drag handle, duplicate button, and delete with confirmation. Clean, unobtrusive overlay UI that stays out of your way.

↩

Undo / Redo

Full history stack. Ctrl+Z to undo any change — text edits, moves, deletes, style changes. Ctrl+Shift+Z to redo. History count shown in the toolbar.

⬇️

Export & Save

Download the edited page as clean HTML with all editor artifacts stripped out. Copy to clipboard. Export a JSON diff of exactly what changed between original and edited.

Get Started in Seconds

Three ways to add n3ware to your project.

1. CDN (quickest)

Recommended 
<!-- Paste before </body> -->
<script src="https://n3ware.onrender.com/n3ware.js"></script>

2. Self-hosted

# Download the file
curl -o n3ware.js https://n3ware.onrender.com/n3ware.js

<!-- Reference locally -->
<script src="/path/to/n3ware.js"></script>

3. JavaScript API

// After including the script:
n3ware.enable();   // activate editor
n3ware.disable();  // deactivate
n3ware.toggle();   // toggle
n3ware.export();   // download HTML
n3ware.undo();     // undo last change

Keyboard Shortcuts

Toggle editor
Ctrl Shift E
Undo
Ctrl Z
Redo
Ctrl Shift Z
