🚀 Getting Started

Dashboard screenshot coming soon

What is n3ware?

n3ware is a visual website editor that lets you build and update your own website without any coding. One click and you're editing — click on text to change it, drag sections to reorder them, drop in pre-built components, and save changes that go live instantly. Your site lives in the cloud; we handle the hosting, CDN, and SSL.

Creating an account

No password needed. Enter your email at n3ware.com → we send you a magic sign-in link → click it → you're in. Your session stays active for 7 days.

The dashboard

After signing in you'll see your Dashboard — the control center for all your sites. From here you can:

  • Create a new site
  • Open an existing site in the editor
  • Manage billing and subscriptions
  • Set up analytics and third-party integrations

Creating your first site

  1. Click New Site in the dashboard
  2. Give your site a name
  3. Pick a template from the gallery, or start with a blank page
  4. Click Open in Editor — your site loads in a new tab with the visual editor ready

✏️ The Editor

Editor screenshot coming soon

Entering edit mode

When you open your site through the dashboard, it loads with the editor active. If you ever need to toggle it: press Ctrl+Shift+E (Windows/Linux) or Cmd+Shift+E (Mac), or click the red n3 button in the top-left corner of the page.

While in edit mode, a thin red border appears around editable elements when you hover over them.

The toolbar

The black toolbar appears at the top of the page in edit mode:

The FAB (floating action button)

The circular button in the bottom-right corner opens a stack of action pills when clicked. Each pill opens a different panel or mode:

Editing text

Click any text element on the page — a heading, paragraph, button label, anything. The element becomes editable and a floating formatting toolbar appears above it. From this toolbar you can:

  • Make text bold or italic
  • Add a hyperlink
  • Change heading level (H1–H6 or paragraph)
  • Change font size
  • Change text color
  • Clear formatting

Press Escape or click outside the element to finish editing.

Selecting blocks

Click anywhere on a section, card, image, or other block element. A control bar appears at the top of the element showing:

  • The element's tag name (e.g., section, div, img)
  • A drag handle to reorder the element by dragging
  • Move up / Move down arrows
  • Duplicate — make an exact copy
  • Delete — remove the element

🧩 Components

What are components?

Components are pre-built, fully-styled page sections built with Tailwind CSS. Think hero banners, pricing tables, testimonial grids, contact forms, team rosters — everything a small business website needs, already designed and ready to drop in.

Opening the component library

Click the FAB (bottom-right) → Components pill. The component panel slides in from the right.

Browsing by category

Components are organized into categories:

Heroes · Pricing · Testimonials · Contact · Menu · Services · Team · Gallery · FAQ · CTA · Stats · Navigation · Dynamic

Click a category to filter. Use the search bar at the top to find by keyword.

Placing a component

  1. Find a component you like and click +
  2. The panel closes and you enter placement mode — the page dims and drop zones appear between existing sections
  3. Click a drop zone to insert the component there
  4. The AI customize modal appears

The AI customize modal

After placing a component, a modal asks you to describe what the section should say and do. You can:

  • Type a description: "Our pizza restaurant in Providence, RI, called Ember & Oak — highlight our wood-fired oven and Tuesday pizza night specials"
  • Optionally upload reference images (your logo, photos of your location, etc.)
  • Click Generate — Claude rewrites the component with your content

Click Skip to use the component's default placeholder text and fill it in manually.

🎨 Theme & Branding

Opening the theme panel

FAB → Theme pill. The theme panel slides in from the right.

Colors

Three brand colors drive your entire site's appearance:

  • Primary — your main brand color. Used for buttons, highlights, links, and key UI elements.
  • Secondary — a complementary color. Used for backgrounds, cards, and supporting elements.
  • Accent — a pop color. Used for badges, tags, and emphasis.

Click any color swatch to open the color picker. You can drag the picker, use the hue slider, or type a hex code directly (e.g., #E31337). Changes apply to the live page instantly.

Branding

  • Logo — upload your logo image (PNG or SVG recommended). It replaces the text wordmark in your nav and footer.
  • Favicon — the small icon that appears in browser tabs. Upload a square PNG (32×32 or 64×64 pixels works well).

Both accept a URL (paste a link to your hosted image) or a direct file upload.

Fonts

  • Heading font — applied to all H1–H6 elements on your page
  • Body font — applied to paragraphs and general text

Choose from a curated selection of Google Fonts. Changes load the font and apply it instantly.

Font sizes

Sliders for H1 through H6 and body text. Values are in pixels. The default heading scale (H1 = 60px down to H6 = 18px) follows typographic best practices — adjust to match your brand's visual style.

Saving your theme

Click Save Theme at the bottom of the panel. Your theme is saved to your site's cloud storage and applied to all pages.

🧭 Navigation

How the nav works

Your site has a fixed navigation bar at the top. It includes your logo/brand name on the left, menu links in the center, and a CTA button on the right. On mobile it collapses into a hamburger menu.

Opening the nav editor

FAB → Nav pill (or select the <nav> element and click the nav edit button in the control bar). The nav editor panel slides in.

Brand settings

  • Logo text — the text wordmark shown when no logo image is uploaded
  • Logo image — upload an image to replace the text
  • Logo link — where clicking the logo goes (usually /)

Menu links

Your nav links are listed in order. For each link you can:

  • Edit the label (the visible text)
  • Edit the href (the URL it links to)
  • Set target (_self for same tab, _blank for new tab)
  • Drag to reorder
  • Click × to delete

Click + Add link to add a new menu item.

CTA button

The right-side call-to-action button. Toggle it on/off, change the label, and set the destination URL.

Sub-navigation

Some page templates include a sticky sub-navigation bar that highlights the current section as you scroll (scroll-spy). This is separate from the main nav — it's an in-page navigation tool for long single-page sites. It auto-discovers <section id="..."> elements and generates links automatically.

🖼️ Images

Image replace modal screenshot coming soon

Replacing an image

Click any <img> element on your page. The element control bar appears, and a Replace image button is shown. Click it to open the image modal.

Using a URL

Paste any publicly accessible image URL — from Unsplash, your own CDN, Google Drive (shared link), or anywhere else. The image loads immediately so you can preview it before confirming.

Uploading a file

Drag and drop an image file onto the upload area, or click to open a file browser. Supported formats: JPG, PNG, GIF, WebP, SVG, AVIF. Max 5 MB per file.

On real hosted sites, the file is uploaded to cloud storage and you get a permanent URL. On the demo (n3ware.com/demo), images are embedded as base64 data URLs in the exported HTML.

Alt text

Every image should have descriptive alt text — a short description of what the image shows. Alt text:

  • Is read aloud by screen readers for visually impaired users
  • Helps search engines understand your images (good for SEO)
  • Displays as fallback text if the image fails to load

Example: instead of "photo1.jpg", write "Chef Marcus Reid plating a wood-fired margherita pizza at Ember & Oak."

What are collections?

Collections let you manage structured, repeatable content — things like team members, menu items, blog posts, or testimonials. Each collection has a schema (the shape of each item — what fields it has) and entries (the actual data).

Instead of hard-coding your team members' names directly in the HTML, you manage them in the Content panel and your page template renders them automatically. Add a new team member → it appears on your page instantly.

Opening the Content panel

FAB → Content pill.

Creating a collection

AI path — just describe it

  1. Click + New Collection
  2. Make sure the "Describe it" tab is active
  3. Type what you need: "team members with name, role, headshot photo, and a short bio"
  4. Click ✦ Create with AI — Claude generates a field schema for you
  5. Review the fields, adjust if needed, then click Create Collection

Preset path — pick a template

  1. Click + New Collection
  2. Click the Pick a preset tab
  3. Choose from: Team Members, Blog Posts, Menu Items, Services, Testimonials, Locations, Portfolio, FAQ, Events, or Products
  4. Review the pre-filled fields, adjust the collection name, then Create Collection

Manual path

  1. Click + New Collection
  2. Type a name and slug
  3. Click + Add field to build your schema field by field
  4. Set the label, type (text, image, number, boolean, etc.), and whether the field is required
  5. Click Create Collection

Adding entries

After creating a collection, click it in the Content panel to open the entry list. Click + Add Entry. An auto-generated form appears with one input per field. Fill in the fields and click Save.

AI sample generation

After creating a collection, n3ware offers to generate sample entries automatically. Type a brief description of your business and how many samples you want (up to 10), then click ✦ Generate. Claude fills in realistic data based on your business context — names, descriptions, prices, even Unsplash photo URLs.

Dropping a collection onto your page

  1. Open the Components panel (FAB → Components)
  2. Click the Dynamic category
  3. Your collections appear as cards — click + on the one you want
  4. Placement mode activates — click where you want the section
  5. A default template block is inserted with {{#each team}}...{{/each}} syntax
  6. The template is replaced with real entry data when visitors view your page

How the template syntax works

When you save a page containing {{#each team}}...{{/each}}, the assembler (the part of n3ware that builds pages for visitors) replaces those markers with real data. {{this.name}} becomes each team member's name, {{this.role}} becomes their role, and so on. You never need to edit this syntax manually — it's generated for you when you drop a collection.

Free tier limits

Free accounts include 2 collections with up to 10 entries each. Upgrade to Pro for unlimited collections and entries.

📁 Media Library

Media library screenshot coming soon

What the media library shows

The Media Library lists every image and file you've uploaded to your site — with file name, size, upload date, and a thumbnail. It's also your place to clean up unused files.

Opening the media library

FAB → Media pill.

Viewing file usage

Each file in the library shows which pages and sections it's being used in (if any). This prevents you from accidentally deleting an image that's currently on a live page.

Deleting media

Click the trash icon on any file. If the file is in use, a warning tells you where it's being used and links you to those pages so you can replace it first before deleting.

Uploading new files

Click Upload in the media panel or drag files directly into the panel. Files are stored in your site's cloud storage and immediately available to use anywhere on your site.

💾 Saving & Publishing

Save button screenshot coming soon

How saving works

When you're happy with your changes, click the green Save button (bottom-right, next to the FAB). Your edits are written to Google Cloud Storage, the assembler cache is cleared, and your live site updates — usually within a few seconds.

What gets saved:

  • Page HTML — the main content of your page
  • Navigation — your menu links, logo, and CTA button
  • Theme — colors, fonts, and sizing
  • Components — header, nav, and footer sections

Undo / Redo

Every edit is tracked in a history. Press Ctrl+Z to undo and Ctrl+Shift+Z to redo. The history resets when you save or reload the page.

Exporting your HTML

Click Download in the toolbar to export your page as a clean, standalone HTML file with Tailwind CDN included. This file opens in any browser without n3ware. You can also click Copy HTML to copy the source to your clipboard.

🌐 Domains & Going Live

Domains screenshot coming soon

Your free build URL

Every site you create on n3ware gets a permanent build URL:

https://n3ware.com/sites/{your-site-id}

This URL always works — no subscription required. It's where you build, test, and share previews. Share it with a client, test on your phone, or use it as a staging environment.

Going live on a custom domain

To make your site available at yourbusiness.com, you need the Pro plan ($20/mo). With Pro active:

  1. Go to Dashboard → your site → Settings
  2. Enter your domain name
  3. Follow the DNS instructions (we'll tell you exactly what record to add at your domain registrar)
  4. n3ware handles SSL (HTTPS) and serves your site through Cloudflare's global CDN

Pointing an existing domain

If you already own yourbusiness.com, add a CNAME record pointing to assembler.n3ware.com. We'll provision an SSL certificate automatically (usually within a few minutes).

❓ Frequently Asked Questions

Can I use n3ware on my existing website?

Yes. Paste your website's URL into the "Try it with your site" box on the homepage. n3ware loads your site in the visual editor so you can experiment with it — no installation required. Save and we'll host the updated version.

Do I need to know HTML or CSS?

No. The editor is entirely visual — click to edit, drag to rearrange, drop to insert. No code required. If you do want to get into the code, the Download feature exports clean HTML you can open in any editor.

Can I edit my site from my phone?

Yes. The editor works on any device with a modern browser. The FAB and toolbar are touch-friendly, and the component panels are scrollable on mobile.

What happens to my site if I cancel my Pro subscription?

Your site and all your content are preserved. Your build URL (n3ware.com/sites/{id}) keeps working. Your custom domain (yourbusiness.com) stops resolving until you resubscribe. Nothing is deleted.

Can I export my site to host it elsewhere?

Yes. Click Download in the toolbar to get a standalone HTML file with Tailwind CSS included. This file works in any browser and can be hosted on any static hosting provider (Netlify, Vercel, GitHub Pages, etc.).

Is my data safe?

Your site files are stored in Google Cloud Storage with object versioning enabled. Every save creates a new version — you can roll back to any previous version from the dashboard. We use Cloudflare for CDN and DDoS protection.

Can I have multiple sites?

Yes. Each site you create is independent, with its own pages, content, and billing. You pay $20/mo per live site — inactive sites are free to keep and edit.
